# LoginWithFirebase

A complete Firebase authentication implementation in Android with Kotlin, demonstrating login, signup, and session management.

## Features

- Email/Password authentication
- Google Sign-In integration
- Password reset functionality
- Session management
- Input validation
- Clean login/signup UI

## Tech Stack

- **Language:** Kotlin
- **Auth:** Firebase Authentication
- **UI:** Material Design / XML Layouts

## Getting Started

1. Clone the repository
```
git clone https://github.com/bhartendukodes/LoginWithFirebase.git
```
2. Open in Android Studio
3. Add your `google-services.json` from Firebase Console
4. Enable Email/Password and Google auth in Firebase Console
5. Build and run
